# goodEvent后台管理系统


## 项目介绍

技术栈：html + css + javaScript + jQuery + Layui + ajax + Node.js

主要实现管理系统的文本管理功能、和个人中心信息配置功能、更多功能具体如下：

layout实现页面布局的搭建、Ajax实现登录注册验证功能、用Layui实现账号密码正则的验证、Ajax实现用户信息的获取功能、jQuery+Ajax实现头像昵称的渲染、complete实现简单的防止无密码登录，等等功能
